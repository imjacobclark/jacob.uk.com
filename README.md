# jacobclark.xyz

A universal React app powering [jacobclark.xyz](https://www.jacobclark.xyz).

# Stack

* Node.js
* React
* Redux
* Express
* Webpack
* Babel
* Jest
* Supertest

## Versions:

* Node version: 9.0.4
* npm 5.6.0

# Building

```shell
$ npm install
$ npm run build
$ npm start
```

[localhost:3000](http://localhost:3000)

# Testing

```shell
$ npm install
$ npm test
```
